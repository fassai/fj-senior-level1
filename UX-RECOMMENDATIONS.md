# FJ Senior Level 1 — UX Recommendations for Seniors
> Saved for future Notion page. Post-MVP reference.

## 1. Too Many Colors → Single Accent
- **Problem**: Red/Yellow/Green/Blue category dots + Orange brand + Blue check-in header = 7 colors competing
- **Fix**: Use only FJ Orange as accent. Drop category color system entirely.
- **Why**: Seniors don't think "Strength day vs Balance day" — they think "Day 4"

## 2. No Clear Day Label → "วันที่ X" First
- **Problem**: Cards show Thai titles with tiny category labels — hard to find which day
- **Fix**: Lead with "วันที่ 1" big and bold. Thai title becomes subtitle.
- **Why**: Day number is the anchor — users need to know where they are in the 30-day journey

## 3. Too Many Numbers → Show Only Essentials
- **Problem**: RPE 2-3/10, 8 ครั้ง, 1 เซต, พัก 45 วินาที, 6/30 วัน, 20%
- **Fix**: Remove RPE from day view. Combine reps+sets into one line. Progress bar alone is enough.
- **Why**: Number overload causes cognitive fatigue. One number per context is the rule.

## 4. English/Jargon → All Thai
- **Problem**: "Strength", "Balance", "Mobility", "RPE", "Retest", "Chair Stand Test", "DOMS"
- **Fix**: All labels in Thai. Remove English category labels. "Retest" → "วัดผลประจำสัปดาห์"
- **Why**: Target users are Thai seniors. English medical terms add confusion with zero benefit.

## 5. Day Detail Overwhelm → Progressive Disclosure
- **Problem**: Header → Banner → Why → Video → Exercises → Check-in → Reflection → Button — all stacked
- **Fix**: Show only: Day title → Exercises → Button. Collapse "why" + video into "อยากรู้เพิ่ม" toggle.
- **Why**: Seniors need to see "what do I do" immediately. Extra info available but not in the way.

## 6. Week Tabs → Simpler Labels
- **Problem**: Theme names like "ปลุกกล้ามเนื้อ" add cognitive load to navigation
- **Fix**: Just "สัปดาห์ 1", "สัปดาห์ 2" — no themes in tabs
- **Why**: Theme names are meaningful to trainers, not to seniors navigating

## Future Improvements (Post-MVP)
- Add FJ logo to header/consent screen
- Encouraging emoji/message on day completion (confetti moment)
- Streak counter ("ทำติดต่อกัน 5 วัน!")
- Font size toggle (ใหญ่/ปกติ) for accessibility
- Audio instructions option for exercise guidance
- Caregiver mode: share progress link with family member
